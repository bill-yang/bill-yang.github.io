how to completely remove flux from linux

When I upgraded my computer to mx linux 21, fluxbox windows manager does not work properly. I run commands to completely uninstall the fluxbox and related packages.

# remove packages
sudo apt purge mx-fluxbox mx-idesktool mx-dockmaker mxfb-art mxfb-docs mx-fluxbox-about mx-fluxbox-data faba-icon-theme moka-icon-theme wmalauncher lxappearance tint2 rofi rofi-calc gkrellm fluxbox idesk 
#Remove config dirs and files
sudo rm -r  {/etc/skel,$HOME}/{.fluxbox,.gkrellm2,.idesktop,.config/{tint2,rofi}}
rm $HOME/.idesksrc
#Remove leftover packages
sudo apt autoremove
